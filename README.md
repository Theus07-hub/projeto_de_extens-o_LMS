Estou aqui para apresentar um projeto com um amigo de faculdade meu, estamos produzindo um sistema LMS de estudantes, a seguir, vou mandar as regras que o projeto deva ter abaixo:





\\## Regras do Projeto...



* Distribuir as disciplinas com carga horária pertinente ao curso e mostrar uma para exemplo;
* Ter no mín 05 alunos listados, que contenham: Nota, Frequência e com resultado de Aprovado ou Recusado;
* Ter um mín de um vídeo aplicado em cada disciplina escolhida;
* Um Quiz com até 05 questões que tenha feedback para acertos e erros;
* Possuir um texto que seja livre mas esteja ligada a disciplina escolhida;
* Ter um fórum com data e horário no calendário com temática determinada e Moderador;
